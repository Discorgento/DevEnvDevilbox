## Installation 

### Update the system

```
sudo apt update && sudo apt upgrade && sudo apt autoremove && sudo apt autoclean
```

### Now run this command to start the installation

```
cd ~ && wget -c https://git.io/J3Bw2 && chmod +x devenvdevilbox.sh && ./devenvdevilbox.sh
```
